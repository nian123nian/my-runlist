# 项目名称：飞飞跑单平台（Vue2）
> 一个移动端适配的跑单平台前端页面，实现了快捷下单、订单管理、用户注册、登录等核心功能。本项目用于展示前端开发能力，可作为作品集参考。
> 
## 技术栈
>前端：Vue2+ElementUI
>后端：springboot2+Mybatis
>小程序：uni-app
>数据库：MySQL
>http请求：Axios + 请求拦截器封装

## 核心功能
管理员：登录、订单管理、联系人管理、认证信息管理、收支信息管理、评价管理、公告管理、管理员管理、用户管理、个人中心、修改密码
骑手：注册、登录、接单（需提交认证骑手）、查看骑手订单
用户：注册、登录
    首页：查看公告、分类下单
    下单页：设置取货地址、收货地址、设置订单信息、提交订单
    订单页：查看订单、取消订单、确认收货、评价订单、删除订单
    我的：头像、姓名、（骑手信息）、我的地址管理、收支明细查看、我的评价管理、我的钱包、充值、个人信息管理、平台介绍、用户协议、退出登录
